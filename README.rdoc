== Campaign Finance Demo

This is a skeleton Rails 3.1 application for demonstrating the Campaign Cash and Fech gems from The New York Times. To get started, clone this repository, then do:

  * bundle install
  * rake db:create
  * rake db:migrate
  * get a NYT Campaign Finance API key from http://developer.nytimes.com/
  * set nyt_api_key in app/controllers/application_controller.rb
  * rails server
  * open http://localhost:3000/fec_filings/
